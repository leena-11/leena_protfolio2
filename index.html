Vehicle Routing using Road Width Detection
Machine Learning & Spatial Routing

This project solves the classical Vehicle Routing Problem (VRP) by taking physical constraints like road widths into account. By integrating computer vision algorithms, the system reads aerial or street images to calculate real-time road widths, preventing oversized logistics vehicles from routing through restricted zones.

Key Features:
  • Automated street width estimation via edge-detection methodologies
  • Constraint-based route optimization with heuristic pruning
  • Interactive visualization map detailing forbidden corridors
Tech Stack:
Python OpenCV NetworkX Google Maps API
